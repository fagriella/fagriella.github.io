# Panduan Upload Arsip & Materi Kuliah

Selamat datang di panduan upload F.AGRIELLA! Panduan ini dibuat khusus untuk memandu kamu (PJ Mata Kuliah) memasukkan file materi, absen, atau foto ke dalam website kelas kita.

Ada **3 cara** yang bisa kamu pilih, mulai dari yang paling gampang banget sampai yang agak teknis. Yuk kita pelajari!

---

## CARA 1: Pakai Formulir Web (Paling Gampang & Disarankan)
Ini cara yang paling sering dipakai. Kamu tinggal isi form di web ini, dan file-nya akan otomatis masuk ke Google Drive kelas kita. Habis itu, datanya langsung muncul di website!

### Langkah-langkahnya:
1. **Pilih Jenis Upload:**
   - **Materi Perkuliahan:** Kalau mau upload PDF, PPT, Word, atau Gambar pelajaran.
   - **Foto Dokumentasi:** Kalau mau upload foto papan tulis atau foto bareng (tenang, fotonya otomatis dikecilin biar web kita nggak lemot).
   - **Tugas Kuliah:** Kalau ada PR atau tugas. Nanti otomatis **semua anak kelas bakal dapet notifikasi** di HP mereka!
   - **Hapus Arsip:** Kalau kamu salah upload dan mau hapus file.
2. **Pilih Semester & Mata Kuliah:** Disamain aja dengan kelasnya.
3. **Nama File atau Link (Boleh Dikosongkan):** 
   - Kosongin aja kalau mau file-nya pakai nama asli.
   - **Tips:** Kalau misal materinya itu Video YouTube atau Link Google Drive lain, kamu bisa masukin link-nya ke kolom ini, tanpa perlu upload file beneran!
4. **Lampirkan File:** Masukin file kamu ke kotak yang disediain (Maksimal 50MB ya).
5. **Konfirmasi Token:** Masukin token rahasia kamu untuk konfirmasi.
6. Klik tombol **"Kirim Arsip"**. Selesai deh! Nanti datanya langsung masuk web saat itu juga.

---

## CARA 2: Upload Langsung Lewat Google Drive
Cara ini cocok misal kamu punya **banyak banget file** yang mau di-upload sekaligus (borongan). Jadi nggak capek upload satu-satu di web.

### Syaratnya:
Kamu harus minta akses ke admin dulu supaya email kamu bisa edit folder **"Arsip Kuliah"** di Google Drive kelas kita.

### Aturan Taruh File:
Pastikan kamu naruh filenya ke folder yang benar di dalam Drive, biar kebaca sama website. Formatnya gini:
- Buka folder **Semester X**, lalu buka folder **Nama Mata Kuliah**.
- Cemplungin file-nya di situ.
- Contoh: Berarti file-nya ditaruh di dalam `Semester 1 / Pengantar Agribisnis / (file kamu di sini)`

---

## CARA 3: Upload Lewat GitHub (Khusus Foto Besar)
Cara ini lumayan ribet, khusus buat kamu yang ngerti IT atau kalau kamu mau upload **Foto** tapi cara pertama gagal.

### Syaratnya:
Kamu wajib punya akun GitHub dan minta Admin jadikan kamu *collaborator* (rekan kerja).

### Aturan Taruh File:
Kalau mau upload foto makrab atau kegiatan:
- Taruh di dalam folder `arsipfoto / Nama Album Kamu`
- Contoh: Di dalam `arsipfoto / Dokumentasi Makrab / foto_baru.jpg`

---

### Cara Sinkronisasi Biar File-nya Muncul (Penting buat Cara 2 & 3)
Kalau kamu pakai **Cara 2** atau **Cara 3**, file kamu nggak akan langsung seketika muncul di web. Tapi tenang aja:
- **Otomatis:** Website kita bakal nge-cek file baru setiap **30 Menit** sekali secara otomatis.
- **Manual (Cepat):** Kalau nggak sabar nunggu 30 menit, kamu tinggal **Refresh / Muat Ulang (F5)** halaman website F.AGRIELLA kamu. Habis di-refresh, file-nya langsung muncul!

## Solusi Kalau Ada Masalah
1. **Token Salah:** Pastikan kode tulisan Token PJ-nya sudah benar dan nggak ada typo pas ngisi Form (Cara 1).
2. **File Nggak Muncul:** Coba **Refresh / Muat Ulang** web kamu. Kalau masih nggak muncul buat yang pakai Cara 2/3, pastiin nama foldernya udah ejaan yang benar sama dengan di website.
3. **Pesan "File too large / Kegedean":** File maksimal 50MB. Kalau kegedean, upload aja file aslinya ke Google Drive pribadi kamu, lalu salin/kopi link Google Drivenya ke form "Teks Link / Nama File" di Cara 1. Tipe kontennya pilih "Materi Perkuliahan".
